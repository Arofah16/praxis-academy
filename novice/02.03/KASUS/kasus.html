Persamaan dan Perbedaan antara 3 Lybrary (Ramda, Underscore, Lodash)

  • persamaan
  1. ketiganya merupakan alat sumber terbuka
  2. digunakan untuk para programmber yang mempunyai kemampuan di Javascript
  3. digunakan untuk memudahkan penggunaan fungsi yang sangat banyak di Javascript , karena dengan ketiga library tersebut fungsi didalamnya lebih praktis
  4. ketiganya digunakan untuk mempermudah para programmer menyusun kode mereka, terlebih untuk peggunaan function disana
  • perbedaan
  1. penggunaan underscore lebih banyak digunakan dari pada penggunaan library ramda js
  2. ramda lebih spesifik dirancang untuk pemrograman fungsional , dengan tujuan tidak mengubah data yang telah dimasukkan pengguna
  3. underscore digunakan untuk mempersempit objek ketika seseorang sedang mebuat suatu program,fungsinya lebih pada pembuatan kerangka javascript , pembuatan indeks dengan lebih cepat, serta pengujian kesetaraan yang lebih endalam
